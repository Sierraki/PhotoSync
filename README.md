# PhotoSync


一款跨平台的手机相册同步工具。支持通过 **局域网 (Wi-Fi)** 或 **ADB (有线)** 连接，将手机端的照片与视频高速同步至电脑端。

## 主要特性
* **多模连接**：支持局域网链接与 ADB 高速有线传输。
* **增量同步**：智能识别已同步文件，仅传输新增照片。
* **灵活过滤**：支持按日期、相册名称、文件类型（图片/视频）进行筛选。
